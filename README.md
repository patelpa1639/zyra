# Zyra 🏃

AI-powered health & fitness platform. Connect Strava, Garmin, Apple Health and more — get personalized AI insights, recommendations, and team game plans.

## Apps

- `apps/web` — Next.js web app
- `apps/mobile` — Expo React Native mobile app
- `apps/api` — Fastify backend API

## Packages

- `packages/shared` — Shared TypeScript types

## Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion
- **Mobile**: Expo + React Native
- **Backend**: Fastify
- **Database**: Supabase (Postgres)
- **AI**: Claude API (Anthropic)

## Getting Started

```bash
npm install
npm run dev:api   # API on port 3001
npm run dev:web   # Web on port 3000
```
